# CIblog（更新中暂时无法正常使用）
基于tp5和vue.js（2.6.10）开发的博客平台

# 技术栈
前端：vue2+vuex+vue-router+webpack+ES6/7+element-ui
后端：tp5

# 项目运行

`git clone https://github.com/zhangyifei233/CIblog.git`

`cd ciblog/blog`

`npm install`

`npm run serve`
